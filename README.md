# Zomato-Restaurant-Analytics-Customer-Sentiment-Dashboard
This project analyzes Zomato restaurant data to uncover insights related to restaurant performance, customer preferences, ratings, and review sentiment.
The project combines Python, NLP, and Power BI to create an end-to-end data analytics solution.

## Objectives
- Analyze restaurant ratings and votes
- Identify popular cuisines and restaurant types
- Understand online ordering trends
- Perform customer review sentiment analysis
- Build an interactive Power BI dashboard

## Dataset
Source: Kaggle Zomato Restaurant Dataset

Records: 51,000+ restaurants

## Tools & Technologies
- Python
- Pandas
- NumPy
- NLTK (VADER Sentiment Analysis)
- Power BI
- Data Visualization

## Key Insights
- Restaurants with online ordering generally receive higher engagement.
- A small number of cuisines dominate restaurant listings.
- Positive sentiment represents the majority of customer reviews.
- Higher-rated restaurants tend to attract more customer votes.
